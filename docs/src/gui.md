# GUI Guide

Launch with `aaai-gui`.

1. **Opening screen** — select Before folder, After folder, audit definition.
2. **File tree** — all changed files with status badges.
3. **Diff viewer** — side-by-side before/after diff.
4. **Inspector** — enter reason, choose strategy, click Approve & Apply.
5. **Save** — persist the definition.
6. **Re-run Audit** — verify updated rules.
